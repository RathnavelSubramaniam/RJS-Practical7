# RJS-Practical7
Create a Simple React Application Using `React.createElement()`

## Objective
Create a simple React application that displays the message **"Welcome to React"** using the `React.createElement()` method.

## Problem Statement
Develop a React application that creates an `<h1>` element with the text **"Welcome to React"** and renders it inside the HTML element with the id `root`.

## Instructions

- Do modify `index.html` for including the CDN links.
- Write your solution in `index.js`.
- Use the following React methods:
  - `React.createElement()`
  - `ReactDOM.createRoot()`
  - `root.render()`
- Display the text exactly as:
  ```
  Welcome to React
  ```

## Expected Output

```
Welcome to React
```
